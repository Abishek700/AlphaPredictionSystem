This script  include functionalities such as sensor data acquisition, preprocessing, model inference and  real-time interaction with embedded systems.Features¥ Serial Communication: Reads and writes data between a connected Arduino device and a Python script.¥ Sensor Data Handling: Processes real-time data from the LSM9DS1 IMU sensor for motion analysis.¥ Machine Learning Inference : Loads and runs TensorFlow Lite models for embedded AI applications.¥ Data Logging & Visualization: Captures and processes sensor readings for debugging or analysis.DependenciesUsage
1. Connect your Arduino Nano 33 BLE Sense to your PC via USB.2. Upload the corresponding Arduino sketch to the board using the Arduino IDE.3. Run the Python script to start communication:python arduinoidecode.py4. Monitor the output, which may include real-time sensor readings, ML model inference results, or system logs.Configuration¥ Serial Port: Update the SERIAL_PORT variable in the script to match the correct port (e.g., COM3 on Windows, /dev/ttyUSB0 on Linux).¥ Baud Rate: Default baud rate is set to 115200. Ensure this matches the baud rate in your Arduino sketch.Expected Output¥ If processing sensor data, expect real-time acceleration, gyroscope, or magnetometer readings.¥ If using a TensorFlow Lite model, expect classification predictions based on sensor input.¥ If logging data, expect CSV file generation or graphical plots for analysis.Troubleshooting¥ Serial Connection Issues: Ensure the correct port is selected, and the Arduino is properly connected.¥ Data Not Updating: Check the Arduino sketch to ensure it is sending data via Serial.println().¥ Module Not Found Errors: Run pip install -r requirements.txt to install missing dependencies.
