kalmanfilter.applyKalmanFilter ( data,
columns,

Applies Kalman Filter to smooth specific columns in a dataset.

Parameters
dataThe dataset (e.g., pandas DataFrame) containing the columns to be smoothed.
columnsList of column names to which the Kalman Filter should be applied.
paramsDictionary of Kalman Filter parameters, including:
  • "kalmanTransitionMatrices": State transition matrix.
  • "kalmanObservationMatrices": Observation matrix.
  • "kalmanInitialStateMean": Initial state mean.
  • "kalmanInitialStateCovariance": Initial state covariance.
  • "kalmanObservationCovariance": Observation covariance.
  • "kalmanTransitionCovariance": Transition covariance.
Returns
The dataset with smoothed columns.
Exceptions
Raisesan exception if a specified column is missing or an error occurs.
